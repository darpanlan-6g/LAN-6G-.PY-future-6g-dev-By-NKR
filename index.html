Multi-Objective Reward Function

The PPO agent is guided by a carefully weighted reward that prioritizes spectral efficiency while penalizing service degradation.

// Reward Function — Layer 2B
R = 0.4·norm_SINR + 0.3·norm_TP − 0.2·norm_latency − 0.1·HO_penalty
+0.4
norm_SINR

+0.3
norm_TP

−0.2
norm_latency
Latency penalty — must stay below 1ms 6G target
−0.1
HO_penalty
Handover cost — degrades QoE in high-mobility scenarios
Training Pipeline

4-Phase Safe Deployment Strategy

No RL agent is deployed directly to live 6G infrastructure. The phased approach systematically closes the sim-to-real gap.

1
📚

Offline Supervised

Initialize models on historical datasets. Physics models and LSTM predictors learn environmental baselines before any RL exploration.

Historical Data
2
🏗

RL in Static Twin

PPO agent explores freely in the virtualized Static Twin. Risky actions carry no real-world consequence during 10&sup6; training steps.

Risk-Free · 10&sup6; Steps
3
📡

RL in Live Twin

Agent fine-tunes against real-time telemetry from SDR hardware. Actions remain sandboxed — no physical commands issued.

Sandboxed Live Data
4
🚀

Deploy

Once Twin Fidelity and safety thresholds are satisfied, the agent assumes full control of the physical 6G network.

Live Network Control
Implementation

Toolchain Options

Two complementary pathways — choose based on whether you prioritize protocol-stack depth or physical-layer accuracy.

ns-3 + ns3-gym

Standard Academic Pathway
  • Simulator: ns-3 with 5G-LENA / mmWave modules
  • RL Bridge: ns3-gym (OpenAI Gym wrapper)
  • IPC: ZeroMQ + Protocol Buffers (C++/Python)
  • Analytics: TensorBoard + Matplotlib
  • Best for: Layer 2B RL training, MAC/PHY protocol interaction

NVIDIA Aerial + Sionna

High-Fidelity Physics Pathway
  • Ray Tracing: Sionna RT — 3D deterministic channel modeling
  • Platform: NVIDIA Aerial Omniverse Digital Twin (AODT)
  • Acceleration: CUDA / Tensor Cores for LDPC decoding
  • Capability: City-scale RF environments in milliseconds
  • Best for: Layer 2A static twin — SINR heatmaps, THz physics
